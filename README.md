## Get Started

Clone project:
```bash
$ git clone https://github.com/JoshuaHernandezAl/BanPay
```
# Backend
## Steps to run the project
1. Navigate to banpay
2. Install dependencies
```bash
$ yarn install
```
3. Only if you use docker: set-up container. This will create database.
```bash
$ docker compose up -d
```
5. Run project
```bash
$ yarn start
```
Note: WHen you clone this project, you will get a postman collection in banpay/BanPay.postman_collection.json you can import into Postman if you want.
